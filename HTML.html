Part 8: Target

The target attribute specifies where to open the linked document
_self - opens the document in the same window/tab as it was clicked: Visit W3Schools!
_blank - opens the document in a new window or tab: Visit W3Schools!
_parent - opens the document in the parent frame Visit W3Schools!
_top - opens the document in the full body of the window: Visit W3Schools!
